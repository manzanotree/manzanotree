A website for LA-based DJ DOSA to promote the DOSA brand and services. 

Built with:
* Gatsby-powered React frontend
* Express-powered Node backend
* Contentful headless CMS
* PostgreSQL

Features:
* Schedulable lottery where entrants enter a mixtape theme to win a custom mixtape.
* Contact form.
* Photos page with gallery lightbox. Supports mouse, keyboard, and swipe navigation.
